Node Installation.
install nodejs package from the https://nodejs.org/en/download/.

node -v			-> to check the node version
npm -v 			-> to check the node package manager version

Angular Installation.
npm install -g @angular/cli@latest

ng v

create new angular project
>> ng new project-name
				y -> would you like to add angular routing
				select CSS
				
run the project
ng serve

create component
ng g c component-name

create directives
ng g d directive-name

create services
ng g s service-name
