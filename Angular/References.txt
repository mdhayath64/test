**********Requirement******************************
SDK 7
Node > 4

**********create project****************************
cd <Project>
dotnet new sln
dotnet new webapi -n <project_name>
dotnet sln add <project_name/>
dotnet sln list


************run server
dotnet run       -> run the server
dotnet watch --no-hot-reload     -> run the server and reload automically
ng server		 -> run angular

**********Migrations
dotnet ef migrations add <migration_command> -o <path>		-> for first time path is required
dotnet ef migrations add <migration_command>
dotnet ef database update

dotnet dev-certs https --trust       -> add microsoft developer certificate

**********Angualr Commands********************

ng g c nav --skip-tests			-> create component
ng g s <services_path> --skip-tests			-> create services ng g s _services/accounts --skip-tests
ng g interceptor <Intercepter_path> --skip-tests -> creat the intercepter ng g interceptor _interceptors/error --skip-tests
ng g environments                -> environments for the Angular to share variables

**********installing the ssl certificate
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
choco install mkcert
mkcert -install
